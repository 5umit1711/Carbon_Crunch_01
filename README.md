Quick start
1. Clone the repo:
   `git clone https://github.com/5umit1711/Carbon_Crunch_01.git`
2. Enter the project:
   `cd Carbon_Crunch_1`
3. Install dependencies:
   `npm install`
4. Start the server:
   - Production: `node server.js`
   - Dev (auto-restart): `npm run start`
5. Open the UI: http://localhost:8080
